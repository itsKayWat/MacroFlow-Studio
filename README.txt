MacroFlow Studio
===============

A visual automation suite for creating reliable automated workflows using reference points and macro sequences.

Purpose:
- Automate repetitive UI interactions
- Create reliable visual automation scripts
- Capture and verify UI elements
- Generate automated test sequences
- Document software workflows

Components:
1. Reference Capture Tool
   - Capture visual reference points
   - Record screen coordinates
   - Save element screenshots
   - Track UI elements

2. Macro Recorder
   - Create automated sequences
   - Use captured references
   - Configure action parameters
   - Save/load macro configurations

Installation:
1. Clone repository
2. Run: python requirements.py
3. Launch reference-capture.py
4. Create macros with macro-tool.py

Requirements:
- Python 3.8+
- Windows OS
- 1920x1080 resolution (recommended)
- Internet connection for web automation

Use Case Example:
A customer service team needs to verify software functionality:
1. Capture reference points of UI elements
2. Create macro for testing workflow
3. Run automated verification
4. Generate test reports
