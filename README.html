What is MacroFlow Studio?

MacroFlow Studio is a comprehensive visual automation suite that combines reference point capture with macro sequencing capabilities. It consists of two main tools:

  • Reference Capture Tool - For capturing and managing visual reference points
  • Macro Recorder - For creating automated sequences using captured references

Key Features

  • Visual Reference Capture
  • Macro Recording and Playback
  • Web Element Support
  • Coordinate Tracking
  • Screenshot Management
  • Video Recording

Installation

  1. Clone the repository
  2. Run requirements.py to install dependencies: 
    python requirements.py
  3. Launch reference-capture.py to gather reference points
  4. Use macro-tool.py to create automated sequences

System Requirements

  • Python 3.8 or higher
  • Windows Operating System
  • 1920x1080 resolution (recommended)
  • Internet connection for web automation features

Getting Started

  1. Launch Reference Capture Tool
  2. Capture UI elements you want to automate
  3. Open Macro Recorder
  4. Create automation sequence using captured references
  5. Save and run your macro
